# DietPlanner

DietPlanner is an Android-first nutrition planning app built for Solana Mobile users. It combines local, privacy-preserving meal planning with wallet-connected SKR premium feature unlocks.

## Highlights

- Personalized daily meal planning based on goal, height, weight, age, and activity level.
- Food discovery using bundled nutrition references from public-source datasets.
- Educational nutrition explanations with clear medical disclaimers.
- Solana Mobile Wallet Adapter integration.
- SKR token payments for premium features such as AI Chef Pro, Deep Food Intel, and Micro-Coach.
- Local-first privacy model: profile and preferences stay on device.

## Blockchain features

DietPlanner uses Solana Mobile Wallet Adapter for wallet authorization and SPL token payment signing. The app does not custody funds, private keys, recovery phrases, or wallet credentials.

## Medical disclaimer

DietPlanner provides educational nutrition information only. It is not medical advice and does not diagnose, treat, cure, or prevent disease. Users should consult a qualified healthcare professional before making diet changes.
